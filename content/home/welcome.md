# Welcome to FreeBSD!

This site emulates a classic FreeBSD FVWM desktop environment.

## Getting Started

- **Right-click** the desktop to launch applications
- **Double-click** folders to browse the filesystem
- **Double-click** files to open them
- **Drag** window title bars to move windows
- **Drag** window edges to resize

## Try This

Open the terminal (xterm) and type `help` for available commands.
