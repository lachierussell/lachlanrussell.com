# Command Line Tool

A powerful CLI tool for automating development tasks.

## Installation

```bash
npm install -g mytool
```

## Usage

```bash
$ mytool --help
$ mytool init
$ mytool build
```

## Commands

| Command | Description |
|---------|-------------|
| `init` | Initialize a new project |
| `build` | Build the project |
| `test` | Run tests |
| `deploy` | Deploy to production |
